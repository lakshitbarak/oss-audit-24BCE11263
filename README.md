# oss-audit-24BCE11263
Open-Source Software Audit project analyzing Python, its licensing, ecosystem, Linux footprint, and practical Bash scripting implementations.
🧾 Open-Source Audit – Capstone Project

📌 Overview

This repository contains a detailed Open-Source Software Audit Project focused on Python. The project explores the origin, philosophy, licensing, ethical considerations, Linux footprint, ecosystem, and comparison with proprietary software.

It also includes practical Bash scripts demonstrating key Linux and open-source concepts.

---

🎯 Objectives

- Understand the concept of Open-Source Software (OSS)
- Analyze Python as an open-source technology
- Study licensing models like PSF, GPL, and MIT
- Explore the Linux environment and file system
- Implement real-world Bash scripting tasks
- Compare open-source vs proprietary software

---

🐍 Chosen Software: Python

Python is a powerful, easy-to-learn programming language widely used in:

- Web Development
- Data Science
- Artificial Intelligence
- Automation
- Scientific Computing

⚙️ Scripts Included

1️⃣ System Identity Report

- Displays system information (kernel, user, uptime)

🚀 How to Run Scripts
